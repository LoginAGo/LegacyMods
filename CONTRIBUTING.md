# Contributing to LegacyMods

Thanks for your interest in contributing! This document will help you get started.

---

## 🧭 How to Contribute

### Reporting Bugs
Open an issue using the **Bug Report** template. Include:
- What you expected to happen
- What actually happened
- Steps to reproduce
- Screenshots if relevant

### Suggesting Features
Open an issue using the **Feature Request** template. Explain what problem you're solving and why it would benefit the community.

### Submitting Code

1. **Fork** the repository
2. **Create a branch** from `main`:
   ```bash
   git checkout -b feature/your-feature-name
   ```
3. **Make your changes** and commit with a clear message:
   ```bash
   git commit -m "feat: add mod version selector"
   ```
4. **Push** your branch and open a **Pull Request** against `main`
5. Wait for review — the maintainer will respond as soon as possible

---

## 📐 Code Style

- Use clear, readable variable names
- Comment anything non-obvious
- Keep components small and focused
- No console.log statements in production code
